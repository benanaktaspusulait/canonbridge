# Changelog

All notable changes to CanonBridge are tracked here.

## 2026-05-21

- Added chunked batch upload sessions, batch retry/redrive status APIs, scheduled run history, outbox replay recovery, and runtime operations dashboards.
- Added Docker/Testcontainers protocol E2E coverage for 10 mock-backed systems and wired it into CI.
- Hardened production auth/OIDC defaults and Docker Compose environment pass-through.
- Added shared `@canonbridge/tokens` brand tokens and applied them to the website and Mapping Studio.
- Improved website SEO metadata, sitemap/robots, mobile navigation, contact form behavior, missing asset handling, and smoke tests.
- Added Mapping Studio runtime `env.js` support for static Nginx deployments.
- Added proprietary license, security policy, contribution guide, and this changelog.
