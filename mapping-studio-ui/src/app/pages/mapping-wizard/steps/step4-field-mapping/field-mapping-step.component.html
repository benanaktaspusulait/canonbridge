JSONata Functions Reference

Complete reference for all available transformation functions

Tip: Switch a field to Expression mode, then click any pattern below to insert it!
String Functions
$uppercase(str)

Convert string to UPPERCASE

Example: $uppercase("hello") → "HELLO"
$lowercase(str)

Convert string to lowercase

Example: $lowercase("HELLO") → "hello"
$trim(str)

Remove leading/trailing whitespace

Example: $trim(" hello ") → "hello"
$substring(str, start, length)

Extract substring from position

Example: $substring("hello", 1, 3) → "ell"
$replace(str, pattern, replacement)

Replace text pattern

Example: $replace("hello", "l", "r") → "herro"
$split(str, separator)

Split string into array

Example: $split("a,b,c", ",") → ["a","b","c"]
$join(array, separator)

Join array elements to string

Example: $join(["a","b"], ",") → "a,b"
$length(str)

Get string length

Example: $length("hello") → 5
Numeric Functions
$number(str)

Convert to number

Example: $number("42") → 42
$round(num, precision)

Round to decimal places

Example: $round(3.14159, 2) → 3.14
$floor(num)

Round down to integer

Example: $floor(3.7) → 3
$ceil(num)

Round up to integer

Example: $ceil(3.2) → 4
$abs(num)

Absolute value

Example: $abs(-5) → 5
$sum(array)

Sum of array numbers

Example: $sum([1,2,3]) → 6
$max(array)

Maximum value

Example: $max([1,5,3]) → 5
$min(array)

Minimum value

Example: $min([1,5,3]) → 1
Array Functions
$count(array)

Count array elements

Example: $count([1,2,3]) → 3
$append(array, value)

Add element to array

Example: $append([1,2], 3) → [1,2,3]
$sort(array)

Sort array ascending

Example: $sort([3,1,2]) → [1,2,3]
$reverse(array)

Reverse array order

Example: $reverse([1,2,3]) → [3,2,1]
$distinct(array)

Get unique values

Example: $distinct([1,2,2,3]) → [1,2,3]
$map(array, function)

Transform each element

Example: $map([1,2], function($v) $v*2) → [2,4]
$filter(array, function)

Filter elements by condition

Example: $filter([1,2,3], function($v) $v>1) → [2,3]
$reduce(array, function, init)

Reduce array to single value

Example: $reduce([1,2,3], function($a,$b) $a+$b, 0) → 6
Date/Time Functions
$now()

Current timestamp (ISO 8601)

Example: $now() → "2024-01-15T10:30:00.000Z"
$fromMillis(num)

Convert milliseconds to timestamp

Example: $fromMillis(1609459200000)
$toMillis(timestamp)

Convert timestamp to milliseconds

Example: $toMillis("2021-01-01T00:00:00Z")
$millis()

Current time in milliseconds

Example: $millis() → 1705318200000
Object Functions
$keys(object)

Get object keys as array

Example: $keys(obj) → ["a","b"]
$lookup(object, key)

Get value by key

Example: $lookup(obj, "a") → 1
$merge(array)

Merge objects in array

Example: $merge([obj1, obj2]) → merged object
$spread(object)

Convert object to key-value pairs

Example: $spread(obj) → array of key-value pairs
Boolean & Logic Functions
$boolean(value)

Convert to boolean

Example: $boolean("true") → true
$not(value)

Logical NOT

Example: $not(true) → false
$exists(value)

Check if value exists

Example: $exists(null) → false
condition ? then : else

Ternary conditional

Example: status="A" ? "Active" : "Inactive"
Common Patterns & Examples
Concatenate with Space
firstName & " " & lastName

Combines two fields with a space separator

Conditional Mapping
status = "A" ? "ACTIVE" : "INACTIVE"

Map values based on condition

Default Value
email ? email : "no-reply@example.com"

Use default if field is empty

Transform Array of Objects
items.($uppercase(productName))

Map and transform each array element

Filter and Extract
$filter(orders, function($v) { $v.status = "completed" }).orderId

Filter array and extract specific field

Nested Field Access
customer.address.city

Access deeply nested fields

Array Element by Index
items[0].name

Get first element's field

String Manipulation Chain
$uppercase($trim(name))

Chain multiple functions

Sum Array Field
$sum(items.price)

Calculate sum of array field values

Filter by Condition
items[price > 100]

Filter array elements by predicate

Count Filtered Items
$count(items[status="active"])

Count items matching condition

Join Array to String
$join(tags, ", ")

Concatenate array elements with separator
